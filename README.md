# Shopping List (Wishlist Planner)

[Live Demo](https://iulianplop1.github.io/Shopping-List/)

## Why I Built This
I built the Wishlist Planner because I found myself constantly impulse-buying things I didn't actually need, while simultaneously failing to save for bigger goals. I wanted a platform where I could securely track my income and expenses, set strict financial goals, and prioritize the items I want to buy.

The app forces me to link items to specific goals and budgets. It acts as a financial cooling-off period—by the time I've saved enough for an item on my list, I often realize I don't even want it anymore. I built this entirely with vanilla HTML, JS, and CSS to keep it lightweight and instantly responsive.

## Features
- **Wishlist & Priority**: Add items with images, prices, and links, and rank them by priority.
- **Financial Goals**: Set specific savings goals and link your wishlist items to them.
- **Budget Tracking**: Input your income and monthly expenses to see exactly how much you can afford to save each month.
- **Time-to-Afford Converter**: Instead of just showing the price tag, the app calculates exactly how many hours or days you need to work to afford a product based on your financial profile.
- **Secure Backend**: Uses Supabase for fast, secure database storage with Row Level Security.
